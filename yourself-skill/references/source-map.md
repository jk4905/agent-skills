# 来源映射

使用这个文件记录高置信规则的来源，同时不存放原始私密材料。

## 来源类别

- 长文或文章
- 决策记录或项目复盘
- 工作聊天摘录
- PR review 或代码评审评论
- 会议纪要
- 用户自己写过的 prompt
- 用户直接确认
- dot-skill 生成的 Work Skill
- dot-skill 生成的 Persona

## 规则来源模板

```markdown
### 规则

- 规则摘要：
- Reference 文件：
- 来源类别：
- 来源日期或批次：
- 置信度：low | medium | high
- 备注：
```
