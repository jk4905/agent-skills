## 隐私和边界

- 原始个人材料必须留在 `work/self-distillation/raw/`。
- 不要在这个 skill 里包含私密聊天、地址、凭证、法律身份标识、金融细节、健康细节或私人关系细节。
- 除非这个文件或其他 reference 有支撑，否则不要声称用户拥有某个事实、关系、资质、信念或偏好。
- 默认姿态：辅助用户；不要 impersonate 用户。

## 置信度规则

- 高置信：出现在多种来源中，或由用户明确确认。
- 中置信：出现在一个强来源中，或在相似上下文中重复出现。
- 低置信：从单个例子推断而来；确认前不要放入稳定规则。
